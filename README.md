# 我的眼睛就是尺

## 玩法

猴子站在悬崖边，按住鼠标左键让金箍棒伸长，松开后金箍棒倒下变成桥。
长度刚好落在对面平台上就过关，太短或太长都会掉下悬崖。

## 运行

```bash
pip install pygame
python main.py
```

## 文件结构

| 文件 | 职责 |
|------|------|
| `main.py` | 入口 |
| `game.py` | 游戏主循环、状态机、事件处理 |
| `monkey.py` | 猴子角色：移动、掉落、绘制 |
| `staff.py` | 金箍棒：伸长、旋转倒下、碰撞判定 |
| `platform.py` | 平台生成与管理 |
| `settings.py` | 常量配置 |

## 操作

- **按住鼠标左键**：金箍棒伸长
- **松开鼠标**：金箍棒倒下
- **Game Over 后点击**：重新开始
